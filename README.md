# BancosNoSQL

Projeto da cadeira de Banco de Dados 2 com o objetivo fazer operações CRUD com vários modelos de Bancos de Dados NoSQL: Redis (Chave-Valor), MongoDB (Documento), Neo4J (Grafo)


## Como rodar os projetos?

Chave-Valor:
~~~bash
cd Chave-Valor
source run.sh
~~~

Documento:
~~~bash
cd Documento
~~~
